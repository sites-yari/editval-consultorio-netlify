    • 05-03-2014
      Horizonte 2020!

      O novo Programa-Quadro "Horizonte 2020" reúne todo o financiamento no domínio da investigação e da inovação na União Europeia para o período 2014-2020.

      Neste momento, existem mais de 160 convites para apresentação de projetos para os diferentes pilares de atuação (Excelência Científica, Liderança Industrial e Desafios Socais) bem como para outras áreas identificadas, pela União Europeia, como sendo áreas de interesse para a investigação a nível europeu.
       

    • Nome CVTube: plataforma para CV em vídeo

      O CVTube é um projeto de I&D internacional que tem por objetivo o desenvolvimento de uma ferramenta online que pretende facilitar a aproximação entre as oportunidades de emprego e as candidaturas dos jovens.

      Este projeto inovador na área dos recursos humanos, tem a duração de dois anos e nele estão envolvidos sete parceiros (entre os quais está a EDIT VALUE®) provenientes de quatro países.

      Mais informações sobre o CVTube em www.project.cvtuber.eu.
    • Nome Learn2Work: uma nova metodologia

      O Learn2work é um projeto europeu que pretende criar uma ligação entre as oportunidades de trabalho geradas pela indústria e os perfis de educação não-formal, apoiando os jovens no contacto com profissões ligadas à indústria do calçado, tornando-as mais apelativas ao desenvolvimento de carreiras profissionais no setor.

      Para alcançar este resultado, a EDIT VALUE® (enquanto líder do projeto) em conjunto com os outros seis parceiros deste consórcio, encontra-se a adaptar o modelo das escolas de produção dinamarquesas à realidade de Portugal, Itália e Espanha.

      Visite o website do projeto em: www.learn2work.eu.
